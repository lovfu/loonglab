<td >   &amp;nbsp　温家宝总理说，2008年也许是中国经济最困难的一年。本刊记者调查发现，经济下滑的影响正在向装备工业领域蔓延。然而，浮华渐退之后，中国装备的春天其实并不遥远。<br/>
   <br/>　　最近几个月，大规模的企业倒闭风潮在沿海地区蔓延，经济下滑成为经济领域最受人关注的话题。应当地企业邀请，本刊记者前往浙江进行实地调查，了解这次经济下滑对当地装备制造业中小企业的影响。<br/>
   <br/>　　当到达宁波栎社机场时，台风"凤凰"的余威还未散尽，天空中浓重的云层让人感到十分压抑，大风裹着雨滴砸在记者身上，似乎预示了经济下滑的影响还远未结束。<br/>
   <br/>　　"对于我们来说，现在是经济地震，所有的人被压在下面等待救援，等待形势好转。我们要保存实力，如果救援队伍到的时候已经变成尸体就没有意义了。"台州机床协会会长陈华贵的一番话，让人感受到了"经济地震"的破坏力。<br/>
   <br/>　　他提到的"救援"确实在紧锣密鼓地进行着：8月1日，部分纺织品、服装的出口退税率由11%提高到13%；8月4日，人民银行对中小企业融资的主渠道--地方商业银行紧急调增10%的信贷额度。进一步的调控政策正在酝酿，并将根据形势变化相机而动。<br/>
   <br/>　　新劳动法、人民币升值、原材料上涨、能源价格高企带来的经济风暴褪去了"中国制造"耀眼的光环。在单纯追求高增长的发展方式难以为继的情况下，中小制造企业选择一条新的发展道路看来已经不可避免，在某种程度上这就是一场新生的开始。<br/>
   <br/>　　从这个意义上说，2008年的经济下滑其实并不是一件坏事。<br/>
   <br/>　　倒闭如潮<br/>
   <br/>　　在民营经济发达的浙江台州、温州等地，当地企业家历来都信奉"一毛钱饿死、一分钱撑死"的生意经，"薄利多销"是他们屡试不爽的法宝。但从今年年初开始，这一法宝似乎开始失灵了。<br/>
   <br/>　　年初，新劳动法正式实施，中小企业的劳动力成本压力陡增。与此同时，人民币升值、原材料和能源价格上涨的影响始终在加强。在这四种主要因素的共同作用下，中小企业成本压力日积月累，等到中央政府为应对通胀压力而采取紧缩政策的时候，大规模倒闭这层窗户纸终于被点破了。<br/>
   <br/>　　据调查，今年1至5月在广东注销的中小企业达3.5万家，比去年同期增加3成半。上半年珠三角有4000家港企结业。东莞9000多家港商中小企业中，今年以来光是鞋厂已关闭近千家。香港中华厂商联合会的调查显示，近六成受访的港商对未来两三年在珠三角的前景感到悲观或非常悲观。<br/>
   <br/>　　在中小企业最为活跃也最为顽强的浙江，情况也不容乐观。"全省200多万家民企处境微妙，大量中小企业生存面临危机"，这是7月初浙江省紧急报送国家发改委的一份中小企业生存环境调查报告中的内容。浙江省经贸委、中小企业局等部门组成了联合调查组，对10多个行业和数十家企业进行调研剖析，他们得出的这一结论并不乐观。<br/>
   <br/>　　浙江今年一季度进出口情况显示，进出口增幅与去年同期相比均有不同程度下滑，出口增幅分别下降3.4和1.7个百分点，进口增幅则分别下降 12.6和1.5个百分点。许多劳动密集型产业，如羊毛衫、打火机、眼镜等，这些由民资掌控、已占全球市场较大份额的产业群，已被逼至生存临界点。<br/>
   <br/>　　可以看出，受损严重的大多是一些消费品制造企业。与之相比，装备制造领域的中小企业的情况相对要好一些，但随着形势的不断恶化，它们同样不能幸免。<br/>
   <br/>　　根据记者在台州的温岭、玉环等中小机床企业聚集地区的调查，在金切机床领域，这次经济调整对当地机床厂家的影响足以用致命来形容。以玉环县为例，当地产业结构以汽摩配件和铜球阀为主，这两大主流产业的大幅下滑直接影响到机床企业。首当其冲是那些产品种类单一、档次低下的厂家。<br/>
   <br/>　　据当地机床协会透露，如果到2008年底经济形势仍未好转，当地的机床企业产值将会继续下滑10%以上，很多小厂将不得不关闭，或者重组转型。记者在当地很多相对较大的机床企业看到，仓库中堆积的机床成品已经有相当规模，企业平均的第二季度销量比去年同期下降30%-40%，个别企业甚至超过 50%。<br/>
   <br/>　　有资料显示，中小企业的数量占全国企业总数的90%以上，实现了全国工业总产值和利税的60%和50%。由于几乎都是劳动密集型，其提供的就业岗位也占据全国城镇就业总数的70%以上。<br/>
   <br/>　　正因如此，企业倒闭风潮带来的压力已远远超出了经济层面。<br/>
   <br/>　　唇亡齿寒<br/>
   <br/>　　在这次经济下滑中"遇难"的中小企业尽管大多都是中低端产品的制造企业，但是从产业链的角度来看，由于其数量众多，它们的倒闭对于上下游企业的影响仍然是巨大的。装备工业产业链上下游的协作相对其他行业要紧密得多，因此这些中小企业的倒闭对于整个装备工业的影响都是比较显著的。<br/>
   <br/>　　在台州，零配件制造企业数以万计。这些企业规模很小，仅仅依靠低廉的价格为下游的中小机床制造企业供应零配件维生。除了海天等极个别生产高端机床产品的制造企业之外，其他几乎所有的机床企业都面临着产品单一、档次低下，不能有效转移成本压力的困境。因此，弱小的零配件企业就不可避免地受到下游企业的连带影响，景气下降甚至关门歇业。<br/>
   <br/>　　在机床制造企业的下游，汽车、摩托车企业的不景气导致配件企业不景气，而这些企业作为机床厂的直接用户，更是直接导致了机床销量的大幅下滑。<br/>
   <br/>　　为了证实这一点，记者前往国内某重型机床企业调查，在装配车间里，记者发现为国内某著名汽车企业生产的总计2亿元的数台2000吨压力机已经组装调试完毕。据了解，这批压力机年初就应该交货，但由于该汽车企业资金方面的原因始终没有交付使用。<br/>
   <br/>　　就这样，汽车企业的困境沿产业链层层传递，最终达到零配件企业，浙江的中小企业制造集群在经济下滑中惨淡经营。<br/>
   <br/>　　就装备工业本身来说，中小企业的不景气，也会在一段时间后影响到更基础的装备制造企业。在机床领域尤其如此，制造低档中小机床需要大量的重型铣床、镗床等高档设备。由于这些设备的生产周期大多超过1年，重型机床企业受到的影响，差不多要到1年以后才能反映出来。<br/>
   <br/>　　尽管目前受到大飞机、探月工程、核电、京沪高铁等国字号工程的带动，这些骨干装备制造企业没有景气下降之忧，但考虑到这些国字号项目完成后需要国民经济的整体繁荣来回收投资和维持发展，中小企业大规模倒闭如果得不到遏制，这些国字号项目也不可能成为重型机床厂等骨干企业的避风港。<br/>
   <br/>　　"覆巢之下，安有完卵"，蓄势待发的连锁反应让人感受到了经济危机的丝丝寒意。<br/>
   <br/>　　2008是过不去的坎吗<br/>
   <br/>　　在今年年初，温家宝总理曾经说过一句话：2008年也许是中国经济最困难的一年。实际上，这是大多数中国人已经认同的共识。<br/>
   <br/>　　年初的大冰雪灾害中，倒塌的输电铁塔、对电煤望眼欲穿的电厂和困在冰雪中归心似箭的旅客给人们留下了深刻的印象；紧接着汶川的大地震，无数生命灰飞湮灭，悲情感染了神州大地；到了7月份，沿海企业大规模的倒闭风潮引来了中央政治局的密切关注。<br/>
   <br/>　　在这些经济大事件的同时，股市大跌、房市减速以及CPI的连续大幅上涨，都给中国人紧绷的神经不断施加压力。就是在这样的背景下，人们自然很容易接受"2008年是最困难的一年"这样的观点。<br/>
   <br/>　　随着奥运会的结束，"奥运景气"将随之消失。在如今悲观情绪盛行的情况下，下半年的中国经济能否迈过2008这道坎变得更加难以预测。<br/>
   <br/>　　综合记者的这次浙江之行和其他方面的调查结果，装备制造领域中小企业普遍对于未来半年甚至一年的经济状况抱有相当强的消极预期。这种预期与奥运会带来的繁荣似乎并不相称，但这就是事实。<br/>
   <br/>　　2008年，真的是一道迈不过去的坎吗？<br/>
   <br/>　　浮华渐退<br/>
   <br/>　　与目前普遍流行的悲观情绪正好相反，去年底和今年初的时候，人们对于2008年经济的走势预测还普遍相当乐观。<br/>
   <br/>　　对于占据基础地位的钢铁、煤炭和石化等行业，金融机构的分析师们普遍给予了非常高的预期：钢铁方面，国际钢铁需求旺盛，国内钢铁企业成本转嫁能力强，高度的景气将指日可待；煤炭方面，运力紧张和下游需求增长将带来新一轮的景气周期；石化方面，在国际油价高企的背景下，旺盛的需求让人们相信明天一定能更好。<br/>
   <br/>　　作为印证，各个权威机构提供的统计数据也令人印象深刻：2007年大中型钢铁生产企业实现利润1447.38亿元，比上年增长49.54%，创历史最好水平；2007年电子信息行业实现销售收入5.6万亿元，增加值1.3万亿元，双双增长超过18%，产业规模继续在国民经济各行业中位居领先； 2007年国内证券市场的煤炭板块大涨166%，而同期大盘涨幅只有71%……<br/>
   <br/>　　上游产业的繁荣，直接来自于下游企业旺盛的需求，而中小企业正是需求大军中非常重要的一部分。在预期高涨的时期，没有人会对美好的未来产生怀疑。但是，这种令人兴奋的景气周期并没有如分析师们所希望的那样持续下去，从2008年年初开始，经济回落乃至下滑就变得不可避免。<br/>
   <br/>　　突如其来的大冰雪和大地震，让国人和世界认识到了中国的国力并非想象的那样强盛，结构性缺陷仍制约着经济整体的可持续发展。尤其是在汶川大地震中，救灾需要大量特种装备，以及像直升机这样的大型装备。而中国装备制造企业并没有能力满足这些需要，其深层次的缺陷一览无余，高增长带来的所谓"成绩" 也被无情打破。<br/>
   <br/>　　装备工业的最终产品大都是资本品，与固定资产投资高度正相关，是典型的投资拉动型的周期性行业。2006年和2007年，国内城镇固定资产投资完成都超过9万亿元，增幅也双双超过20%。<br/>
   <br/>　　近年来，在追逐高利润的目标指引下，外资、民间资本、地方政府和国有企业这四大投资主力军的热情异常高涨。但是，因为缺乏真正强大的内生能力，"热钱"带动下的装备工业繁荣注定了只能是昙花一现。<br/>
   <br/>　　据统计，目前中国设备投资的2/3不得不依赖进口，其中光纤制造设备的100%，集成电路芯片制造设备的85%，石油化工设备的80%，轿车工业设备、数控机床、纺织机械、胶印设备的70%来自于进口产品。<br/>
   <br/>　　在"热钱"追捧的基础上，有了买来的设备和技术，中国的装备工业就像是打了兴奋剂，在通向"长期景气"的道路上狂奔不止。<br/>
   <br/>　　然而，任何浮华必将褪去，任何虚构的美好也必将回归平凡。<br/>
   <br/>　　国际市场对于"中国制造"的抵制，对低端产品为主的中国产品需求的减少，能源和原材料市场的持续上涨，以及国际产业升级和结构调整都加速了中国出口导向型经济由盛到衰的步伐。<br/>
   <br/>　　和中国出口经济一道下滑的是曾经风光一时的"中国制造"概念。年初开始，不断有媒体和专家开始呼吁"中国制造"的转型，即告别"袜子换飞机"的低端制造，向制造业的核心和高端领域转型。<br/>
   <br/>　　然而，这种转型的代价是相当沉重的。近年来大规模的投资不仅助长了粗放发展的冲动，更透支了未来若干年中国经济的元气。很多企业也认识到了转型的必要和迫切，但由于资金大多变成了固定资产，并且也没有专业技术领域足够的积累，只好维持现状等待大环境的好转。<br/>
   <br/>　　俗话说"冰冻三尺非一日之寒"，经济解冻同样不会在短时间内实现。率先走出困局的，一定是那些眼光长远，不为眼前利益所动，扎扎实实苦练内功的企业。<br/>
   <br/>　　"我们在2003年的时候看到了这个景气周期太长，认为下滑情况会很大，当时就做好了最坏的打算，扩展产品品种来应对（危机）。"浙江省玉环县坎门机床厂的董事长张慧坚向记者说明他应对经济危机的办法，"今年这个形势对我们厂的冲击并不是很大，反倒给了我们一个喘息的机会……我们把原来的产品做一个整合和提升……如果做得好，并且明年年初（经济形势）有一个大的回升，我们就会很平稳地度过这个难关。"说这番话的时候，他的眼神里充满了自信和对未来的憧憬。<br/>
   <br/>　　浮华终究是虚幻的，人们追逐热钱的美梦已经破灭，而对于踏踏实实干实业的人们来说，实业的春天就要来了。<br/>
   <br/>　　专精协作才是正道<br/>
   <br/>　　在黑龙江的齐齐哈尔，中国重型机床行业的领军企业齐二机床的重型装配车间里，记者看到的是一派繁忙的景象。巨大的厂房里有条不紊地堆放着重型机床箱体和各种功能部件，工人们三五成群聚在一起，忙着装配和调试机器。在车间的出口处，重型载重卡车排队等候运输组装完毕的机床和功能部件。而在车间外面，数个正在抓紧建设的重型车间已经封顶，工人们正在紧张地进行内部建设。<br/>
   <br/>　　看过浙江中小企业的情况之后，与眼前的景象对比，记者不禁感叹两者之间的差别竟然如此巨大。<br/>
   <br/>　　在谈到经济下滑对重型机床企业的影响时，齐二机床一位资深的高层管理人员对记者表示，因为有大飞机、探月工程、核电建设、造船和高速铁路等大项目，虽然也受到一些中小企业重型机床订单减少的影响，但国内的重型机床企业在未来若干年里的发展状况还是非常好的。<br/>
   <br/>　　当记者询问如何帮助中小企业走出困局时，这位管理人员给出了自己的建议：走专业配套之路，和产业链上的骨干企业结成紧密的协作关系，共同生存、共同繁荣。在走访多位国内机床行业的资深人士之后，他们给出的建议大多也与此类似。<br/>
   <br/>　　2000年左右，中国的中小机床市场需求非常大，在利润的刺激下很多没有机床制造经验的中小企业盲目介入。而机床属于装备制造业中比较复杂的领域，对工艺和技术的要求很高。这些企业的素质还达不到足够的水平，盲目介入只是为了追逐利润。如今市场萎缩，它们自然只能选择倒闭。<br/>
   <br/>　　对于这些企业来说，走出困局最为现实的方法，就是把内功练好，否则就只能选择退出这个行业。机床产品的市场是非常大的，中小企业必须找准自己应该介入的领域。如果把某一种产品或者功能部件做到中国最好，那么一定会有生存空间。而处于产业核心地位的大型企业集团，有义务通过购买这些中小企业的配套件，以协作的方式共同繁荣。这样做，不仅能够提升配套企业的技术能力和利润水平，也能够帮助骨干企业规避风险，是一举多得的好办法。<br/>
   <br/>　　据此反观浙江当地的中小机床企业，可以发现，在多达上千家的机床企业中，只有不到二三十家的企业能够称得上有一定档次，95%以上的机床厂都是一些小作坊式的装配厂。尽管这些企业有着顽强的生命力，但是在过度竞争和微薄的利润影响下，只有活力是远远不够的。真正把产品做到极致，才是生存发展的正道。<br/>
   <br/>　　在德国和日本，相当多的中小企业都拥有"世界第一"的头衔，在各自的专业领域拥有其他企业无法匹敌的制造技艺。它们几乎都是某个或某些大型骨干主机企业的配套件供应商，经过几十年紧密协作的历史，它们已经成为骨干企业不可或缺的帮手。另外，其中某些实力雄厚的企业，本身就是骨干企业，能够带领其他的中小企业一起发展。<br/>
   <br/>　　从国家整体产业安全的角度来看，这种分工协作也足以使之拥有强大的抵御风险的能力。中国企业目前这种粗放的过度竞争，只会拖累整个产业，使之始终处于利润微薄、弱不禁风的低端领域。<br/>
   <br/>　　在浙江采访的过程中，坎门机床厂的董事长张慧坚向记者介绍，坎门机床厂早在2003年就已经开始拓展产品线。尽管当时企业内外有很多不理解甚至反对的声音，认为错失在单一产品上赚大钱的机会，但他还是坚持走产品多元化和专业化的道路，开发了专用机床、组合机床、立式（卧式）加工中心等中高档机床，现在已经能够生产20多个品种、100多种型号的产品，具有很强的竞争力。<br/>
   <br/>　　在坎门机床厂的装配车间里，记者看到了装配完毕，准备出口到荷兰的数控机床。能得到国际市场的认可，充分证明了专业化和多元化道路的正确。<br/>
   <br/>　　通过此次走访，记者发现，浙江当地几家相对较大的机床企业，正在践行专业化和多元化的道路，接受采访的企业主，也都表示这次经济下滑带来的行业洗牌，对于他们来说是绝对的利好消息，经过这次洗礼浙江机床企业的未来会更好。<br/>
   <br/>　　迎接春天<br/>
   <br/>　　这次经济下滑之所以会引发如此广泛的关注，中央政治局常委们亲赴沿海地区调研是非常重要的原因之一。在调研的过程中，各位常委始终强调"转型" 和"升级"。而在已经公布的下半年经济工作的重点中，中央政府更是明确要求"深化改革开放，着力推进经济结构调整和发展方式转变，提高经济发展质量和效益 "。<br/>
   <br/>　　如何转型？让强者活下来，并且活得更健康长久才是正道。此外，有效疏散这次经济下滑产生的"企业难民"，让它们按照市场法则在产业链上合理配置，寻找自己的生存之道也是转型的应有之义。<br/>
   <br/>　　据《2006～2007年中国装备制造业发展报告》统计，截至2007年底，开发银行向装备制造业企业贷款余额为540亿元。《报告》还认为，中国装备制造业存在自主创新能力弱、关键零部件和关键材料发展滞后、用户对国产装备支持仍不够、政策环境有待改善等诸多问题。<br/>
   <br/>　　为了解决这些问题，与继续支持国家级项目同样重要的，就是帮助中小企业转型。具体地讲，政府应该选择有实力、专业化程度高、经营稳健的中小企业进行补贴，补贴的内容可以包括一般融资和技改、研发补贴。同时，指导和支持骨干企业培育、整合产业链，建立共存共荣的良好企业生态链，实现和谐可持续的发展。<br/>
   <br/>　　特别要提出的是，不论采取何种调控措施，国产化都是必须坚守的基本原则。在"十一五"规划中，振兴装备制造业的目标是进口依赖度从67%下降到30%以下，这既为中小企业提供了巨大的生存空间，也是捍卫民族产业安全，增强国际市场竞争实力的战略举措。<br/>
   <br/>　　冬天来了，春天就一定不会遥远。当告别了浮华的诱惑与困扰，当具有实业精神的人们行动起来，中国制造美好的春天就为期不远了。　<br/>
